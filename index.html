What They Do

Computer Programmers Create, Modify, and Test Code that allow computer applications to work. The work that they are given is mainly from Software Developers and Web Developers.They may develop and write computer programs to store, locate, and get specific data.

They also turn designs and code created by software developers and engineers into code that a computer can follow.

What They Do On The Job

They Anzalize, review, and rewrite programs and code using a workflow chart. They apply knowledge of computer capabilities. They also correct errors by making appropriate changes and then going over the program again to make sure everything is made correctly

The last thing they do is repair or expand existing programs to make sure the website and programs are working at their best potential.
